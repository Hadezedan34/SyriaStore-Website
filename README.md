# SyriaStore — Next.js + Tailwind starter

Minimal Next.js (app router) starter scaffolded with Tailwind CSS and a responsive navbar.

Getting started

1. Install dependencies

```bash
npm install
```

2. Run the dev server

```bash
npm run dev
```

Open http://localhost:3000

What was added

- `app/layout.jsx` — root layout including the navbar
- `components/Navbar.jsx` — responsive navbar with bilingual name (English + Arabic), search, and cart icon
- `styles/globals.css` — Tailwind entry
- Tailwind/PostCSS configs and `package.json` scripts
